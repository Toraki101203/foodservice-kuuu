# モグリス デザイン指示書 — Antigravity 向け

## プロジェクト概要

### サービス名
**モグリス（Moguris）**

### コンセプト
飲食店の「今この瞬間」を、近くにいる人に届ける SNS プラットフォーム。

食べログやホットペッパーのような「検索ツール」ではなく、**Instagram のように好きなお店の日常が流れてくる**体験を提供する。飲食店が Instagram に投稿したコンテンツを、位置情報を使って**近くにいるユーザーに届ける**ことが最大の差別化ポイント。

### ターゲットユーザー
- **一般ユーザー（20〜40代）**: 飲食店好き。行きつけの店のリアルな情報を知りたい。新しい店も見つけたい
- **店舗オーナー**: Instagram で集客している飲食店オーナー。新規客にリーチしたい

### 競合との違い

| 項目 | 食べログ / ホットペッパー | Instagram | **モグリス** |
|------|------------------------|-----------|----------|
| 第一印象 | 「検索ツール」 | 「SNS」 | **「飲食店 SNS」** |
| 情報の鮮度 | 口コミ（過去） | リアルタイム | **リアルタイム** |
| 発見方法 | 検索・地図 | フォロー・おすすめ | **フォロー + 位置情報** |
| アクション | 予約 | いいね | **空席確認 + 予約** |

### 参考アプリ（世界観の参考）
- **Instagram**: クリーンさ、ストーリー、フィードのレイアウト
- **X (Twitter)**: タブ切り替え（フォロー中 / おすすめ）
- **Uber Eats**: マップ上のピン表示
- **食べログ（反面教師）**: 情報密度が高すぎるデザインは避ける

## ブランドアイデンティティ

### トーン & ムード
- **温かみがあるが、洗練されている**
- **食欲をそそる、でもうるさくない**
- Instagram のクリーンさを保ちながら、「食」の温かみを感じる配色

### キーワード
`リアルタイム` `身近` `クリーン` `温かみ` `シンプル` `写真が主役`

### ロゴ
- 「モグリス」テキストロゴ
- フォント: 丸みを帯びたモダンなサンセリフ
- カラー: オレンジ系（プライマリカラーと統一）
- アプリアイコン案も必要

---

## カラーパレット

### 基本方針
- プライマリはオレンジ系（食欲・温かみ）
- 写真コンテンツが映えるように、UIは控えめな色使い
- 真っ黒（#000）は使わない。テキストは gray-700 基調

### 指定カラー

```
プライマリ
├── メイン        orange-500   #f97316
├── ホバー/プレス   orange-600   #ea580c
└── 背景ハイライト  orange-50    #fff7ed

空席ステータス（固定・変更不可）
├── 空席あり      green-500    #22c55e  / 背景 green-50
├── やや混雑      yellow-500   #eab308  / 背景 yellow-50
├── 満席         red-500      #ef4444  / 背景 red-50
└── 休業         gray-400     #9ca3af  / 背景 gray-50

テキスト
├── 見出し        gray-900     #111827
├── 本文         gray-700     #374151
├── サブテキスト   gray-500     #6b7280
└── プレースホルダー gray-400    #9ca3af

背景
├── ベース        white        #ffffff
├── セカンダリ     gray-50      #f9fafb
└── ボーダー      gray-200     #e5e7eb

ストーリーリング
├── 未読         orange-400 → orange-600（グラデーション）
└── 既読         gray-300
```

### デザイナーへの依頼
- 上記は Tailwind CSS のデフォルト値をベースにした指定です
- ブランドカラーとして、オレンジの正確なトーンは調整 OK
- 空席ステータスの色は UX 上の意味があるため変更不可
- 追加のアクセントカラーが必要であれば提案してください

## タイポグラフィ

### フォント
- **日本語**: Noto Sans JP（Google Fonts）
- **ウェイト**: Regular (400) / Medium (500) / Bold (700)

### サイズ体系

| 用途 | サイズ | ウェイト | 備考 |
|------|--------|---------|------|
| ページタイトル | 24px (1.5rem) | Bold | text-balance（自然な改行） |
| セクション見出し | 20px (1.25rem) | Bold | |
| カード内店名 | 18px (1.125rem) | Bold | |
| 本文 | 16px (1rem) | Regular | leading-relaxed（ゆったり行間） |
| サブ情報 | 14px (0.875rem) | Regular | 距離、時刻、ジャンル |
| バッジ/ラベル | 12px (0.75rem) | Bold | |

### 日本語テキスト注意事項
- 見出しは `text-balance`（不自然な1文字改行を防ぐ）
- 本文は `text-pretty` + ゆったりした行間
- 数値データ（距離、件数）は等幅数字で表示

## アニメーション

### 原則
- **最小限**。明示的に必要な場所のみ
- インタラクションフィードバックは 200ms 以内

### 許可するアニメーション
| 要素 | アニメーション | 時間 |
|------|-------------|------|
| ボタンタップ | 少し縮む（scale 0.98） | 150ms |
| ボトムシート | 下からスライドイン | 300ms |
| トースト | 上からフェードイン | 300ms |
| ♡ お気に入り | ポップ（scale 1.2→1） | 200ms |
| ストーリープログレスバー | 左から右にリニア進行 | 5秒/枚 |
| スケルトン | パルス（明滅） | 継続 |

### 禁止
- グラデーションアニメーション
- レイアウトプロパティ（width, height）のアニメーション
- パープル / マルチカラーグラデーション
- グロー効果

## 技術的な制約（デザイナーが知っておくべきこと）

1. **Tailwind CSS** を使用 — デフォルトのスペーシング（4px刻み）とカラーパレットがベース
2. **Next.js Image** を使用 — 画像は必ず固定サイズ or fill + object-cover
3. **Google Maps** 埋め込み — マップ部分のデザインカスタマイズは限定的（ピンのカスタムは可能）
4. **safe-area** — iPhone のノッチ/Dynamic Island 対応が必要
5. **44px ルール** — タップ可能な要素は最小 44 × 44px
